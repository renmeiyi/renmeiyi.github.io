1、类与对象

1、使用变量来管理不利于数据的管理，

2、数组难以体现数据类型，

3、不能体现行为信息。

故引出类与对象（OOP）的概念

一个程序就是一个世界，有很多事物（对象[属性，行为]）

类是一种定义的数据类型，可以是Java定义的也可以是自定义的。

Java定义例如：int，String

对象就是一个具体的实例。

例如类是设计图纸，对象就是根据类具体出的实物。（实物的属性行为，尺寸、颜色等根据设计图纸进行限制，但需要跟设计图纸中的基本保持一致）
